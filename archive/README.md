# Archive

This directory holds legacy docs, duplicate prompt packs, and scratch material that should not participate in primary code or graph discovery.

Rules:
- Keep canonical product/runtime docs at repo root, `.agent/`, `SP500 Predictor/`, `market_sentiment_tool/`, `Weather/`, and `shared/`.
- Move superseded specs, copied prompt bundles, and historical scratch files here instead of leaving them in active discovery paths.
- Do not treat archive content as source-of-truth unless a task explicitly requires historical lookup.
